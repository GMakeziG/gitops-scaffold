# Examples

Sample application definitions for exercising `gitops-scaffold` against, and
(once implemented) for golden-file testing of generated output.

- [`basic/docker-compose.yml`](basic/docker-compose.yml) — a minimal
  single-service web app with a named volume, used as the first fixture for
  Compose parsing in v0.2.

None of these are parsed yet — Docker Compose parsing is the v0.2 milestone.
See [`docs/roadmap.md`](../docs/roadmap.md).
