# Handoffs

Session-to-session handoff notes for `gitops-scaffold`. Each file captures
the exact project state (git commit/tag, completed work, known
limitations, recommended next steps) at the end of a development session,
so the next session can resume without re-reading the full project
history.

Named `YYYY-MM-DD-<milestone>-handoff.md`.
